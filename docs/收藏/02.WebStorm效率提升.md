### 解决不识别 @ 目录

```javascript
// 根目录下新建 jsconfig.json
{
  "compilerOptions"
:
  {
    "baseUrl"
  :
    "./",
            "paths"
  :
    {
      "@/*"
    :
      [
        "src/*"
      ]
    }
  }
,
  "exclude"
:
  [
    "node_modules",
    "dist"
  ]
}
```
