## in操作符
判断是否存在某个属性
```typescript
var obj = {
    keys: ['none','false'],
    ShadowRoot: true
}
console.log('keys' in obj)  // true
console.log('ShadowRoot' in obj)    // true
console.log('name' in obj)  // false
```
