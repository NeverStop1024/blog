## 作用域插槽

使用场景：插槽获取子组件数据

```html
<!--child.vue-->
<template>
  <div class="child-box">
    <p>我是子组件</p>
    <slot name="header" text="我是子组件小猪课堂" :count="1"></slot>
  </div>
</template>
```

```html
<!--App.vue-->
<template>
  <img alt="Vue logo" src="./assets/logo.png"/>
  <child>
    <!-- 可以用解构 -->
    <template #header="{ text, count }">
      <div>{{ text }}---{{ count }}</div>
    </template>
  </child>
</template>
```
